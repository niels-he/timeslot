# timeslot

Is an timeslot booking web app.

## Getting started

Clone the project

### Frontend
1. ```$ cd frontend```
2. ```$ npm i```
3. ```$ npm run start```

#### Run tests
```$ npm run test```

### Backend
1. ```$ cd backend```
2. ```$ npm i```
3. ```$ npm run start```

#### Run tests
```$ npm run test```
